Internet - Como surgiu? voltar

Década 1950 - Guerra Fria

Começou com a DARPA, que realizou os investimentos iniciais essenciais ao desenvolvimento de tecnologias na área de defesa. Além disso, o cientista da computação Joseph Carl Robnett Licklider, considerado o pai do cloud computing, já havia teorizado sobre uma rede onde todos poderiam se comunicar através dela.

A internet é um sistema de comunicação em pacotes, enviados um a um e que permite múltiplos destinos.

Em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web

Como funciona o HTML? voltar

Criada Pelo britânico Tim Berners-Lee, o acrônimo HTML significa Hypertext Markup Language.

Os hipertextos são conjuntos de elementos conectados.

O HTML é uma linguagem de marcação com a qual os elementos de uma página são identificados e definidos, geralmente, por um par de marcadores conhecidos como tags. Esses elementos podem ser desde textos e suas formatações, como parágrafos, sublinhados e links, até conteúdos multimídia, como imagens e vídeos.
